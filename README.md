### 网易云音乐小程序

该小程序仿照网易云音乐小程序开发，能够进行用户登录，音乐搜索和音乐播放等功能

该小程序设计到云开发，需配置云开发环境

###### 下载依赖

```
npm install
```

点击运行项目
